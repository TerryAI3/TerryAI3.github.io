## 🔧 技术要点记录

### 关键发现
1. **图片必须使用HTTP URL**：Docker容器无法访问主机文件
2. **标签数量限制**：小红书限制10个标签，超出的自动截取
3. **超时不等于失败**：30秒超时后后台可能仍在处理
4. **图片下载目录**：`/tmp/xiaohongshu_images/`（容器内）

### 错误处理
```bash
# 常见错误及解决方案

# 1. 图片文件不存在
错误：图片文件不存在: /path/to/image.jpg
解决：使用HTTP图片URL，不要用本地路径

# 2. 标签数量超过限制
错误：标签数量超过10，截取前10个标签
解决：确保tags数组不超过10个元素

# 3. 发布超时
错误：MCPORTER_CALL_TIMEOUT
解决：等待2-3分钟后验证是否发布成功

# 4. 登录状态问题
解决：运行 mcporter call xiaohongshu.check_login_status
```

## 🔄 自动化脚本

### 简化发布脚本
```bash
#!/bin/bash
# /root/.openclaw/workspace/scripts/xiaohongshu_publish.sh

TITLE="$1"
CONTENT_FILE="$2"
IMAGES="$3"
TAGS="$4"

# 读取内容
CONTENT=$(cat "$CONTENT_FILE")

# 发布命令
mcporter call xiaohongshu.publish_content \
  title="$TITLE" \
  content="$CONTENT" \
  images="$IMAGES" \
  tags="$TAGS" \
  is_original=true

# 记录发布
echo "$(date): 发布 $TITLE" >> /root/.openclaw/workspace/小红书发布记录.md
```
