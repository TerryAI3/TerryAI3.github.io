## 📋 标准化参数

### 1. 标题 (title)
- **要求**：最多20个中文字符
- **格式**：吸引眼球+核心关键词
- **示例**：`Cadence总部改造ENR获奖项目`

### 2. 正文 (content)
- **结构**：
  1. 吸引开头 + 表情符号
  2. 核心亮点（使用emoji和分段）
  3. 技术细节/数据支撑
  4. 行业启示/实用建议
  5. 精华总结
  6. 互动话题（3个问题）
- **长度**：300-500字最佳

### 3. 图片 (images)
- **来源**：Pexels高质量免费图片（避免版权问题）
- **数量**：1-3张最佳
- **格式**：HTTP/HTTPS URL数组
- **备用图片**：
  - `https://images.pexels.com/photos/380769/pexels-photo-380769.jpeg`
  - `https://images.pexels.com/photos/1170412/pexels-photo-1170412.jpeg`
  - `https://images.pexels.com/photos/37347/office-sitting-room-executive-sitting.jpg`

### 4. 标签 (tags)
- **数量**：最多10个，自动截取前10个
- **选择策略**：
  - 核心关键词（2-3个）
  - 长尾关键词（3-4个）
  - 流量标签（2-3个）
- **示例**：`["办公空间设计","未来办公","办公室改造","智能办公室","员工福祉","可持续设计","办公家具","空间规划","项目管理","绿色建筑"]`

### 5. 原创声明 (is_original)
- **值**：`true`（声明原创）或 `false`（不声明）
- **建议**：原创内容设置为`true`

## ⚠️ 常见错误处理

### 1. 图片文件不存在
```
错误：图片文件不存在: /path/to/image.jpg
原因：Docker容器无法访问主机文件系统
解决：使用HTTP图片URL，不要用本地路径
```

### 2. 标签数量超过限制
```
错误：标签数量超过10，截取前10个标签
原因：小红书限制最多10个标签
解决：确保tags数组不超过10个元素
```

### 3. 发布超时
```
错误：MCPORTER_CALL_TIMEOUT
原因：30秒默认超时，但后台可能仍在处理
解决：等待2-3分钟后验证是否发布成功
```

### 4. 登录状态问题
```
解决：运行 mcporter call xiaohongshu.check_login_status
```

## 🔄 自动化脚本

### 简化发布脚本
```bash
#!/bin/bash
# scripts/publish.sh

# 参数：标题 内容文件 图片URL数组 标签数组
TITLE="$1"
CONTENT_FILE="$2"
IMAGES="$3"
TAGS="$4"

# 读取内容
CONTENT=$(cat "$CONTENT_FILE")

# 执行发布
echo "正在发布小红书内容：$TITLE"
mcporter call xiaohongshu.publish_content \
  title="$TITLE" \
  content="$CONTENT" \
  images="$IMAGES" \
  tags="$TAGS" \
  is_original=true

# 记录发布
echo "$(date): 发布【$TITLE】" >> ../发布记录.md
echo "发布完成，请等待2-3分钟后验证结果"
```
