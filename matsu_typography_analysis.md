# 玛祖铭立网站 - 排版与字体分析

## 字体系统

### 标题字体
- **英文标题**：使用现代无衬线字体，粗体，大小写混合
- **中文标题**：使用现代黑体，粗体，字重较重
- **特点**：强调感强，视觉冲击力大

### 正文字体
- **英文正文**：现代无衬线字体，中等字重
- **中文正文**：清晰易读的黑体或宋体
- **特点**：清晰可读，专业感强

## 排版布局特点

### 1. 标题排版
- **大小对比**：标题字号非常大（通常是正文的 8-10 倍）
- **行距**：紧凑的行距，增强视觉冲击力
- **字间距**：适度的字间距，增强可读性
- **对齐方式**：左对齐或居中对齐

### 2. 正文排版
- **字号**：中等大小，易于阅读
- **行距**：适度的行距（通常 1.5-1.8 倍行距）
- **段落间距**：充分的段落间距，增强可读性
- **列宽**：适度的列宽，避免过长

### 3. 视觉层级
- **标题**：最大、最粗、最醒目
- **副标题**：中等大小、中等粗细
- **正文**：较小、较细、易读
- **辅助文本**：最小、最细、灰色

### 4. 空间利用
- **充分的留白**：大量的空白空间，增强高级感
- **模块化布局**：清晰的内容模块划分
- **对齐网格**：严格的对齐，体现秩序感

## 色彩与排版的结合

### 强调手法
- **颜色对比**：使用鲜艳的颜色（橙色、绿色）突出重要内容
- **背景色块**：使用背景色块强调标题或重点内容
- **线条装饰**：使用线条、虚线等装饰元素增加视觉趣味

### 视觉引导
- **从上到下**：标题 → 副标题 → 正文 → CTA
- **从左到右**：重要内容优先放在左侧
- **对比度**：高对比度的文字和背景

## 应用到佐迪网站的建议

### 1. 标题排版优化
- 增大标题字号，提升视觉冲击力
- 使用更粗的字重（Bold 或 Black）
- 适度减少行距，增强紧凑感

### 2. 正文排版优化
- 增加行距到 1.6-1.8 倍
- 优化段落间距
- 使用更清晰的字体

### 3. 视觉层级优化
- 强化标题与正文的对比
- 使用颜色和背景色块突出重点
- 添加装饰线条和元素

### 4. 空间利用优化
- 增加留白，提升高级感
- 优化模块间距
- 强化对齐和秩序感

## 具体实现方案

### 字体选择
- **标题**：使用 Oswald 或 Archivo Black（英文），思源黑体 Bold/Black（中文）
- **正文**：使用 Inter 或 Roboto（英文），思源黑体 Regular（中文）

### 字号规范
- **H1 标题**：72-96px
- **H2 标题**：48-64px
- **H3 标题**：32-48px
- **正文**：16-18px
- **辅助文本**：12-14px

### 行距规范
- **标题**：1.0-1.2 倍行距
- **正文**：1.6-1.8 倍行距
- **列表**：1.5 倍行距

### 颜色与排版
- **标题**：深炭灰或建筑蓝
- **强调**：信号橙或建筑蓝
- **正文**：深炭灰
- **辅助文本**：中灰色
