# 佐迪智能 (Zodi Smart) 网站深度整合优化指南

## 1. 视觉排版：玛祖铭立 (Matsu) 风格
- **极简网格布局**：首页采用大面积留白，产品图采用非对称式网格排版，营造“画廊级”的艺术感。
- **动态交互**：滚动时图片轻微缩放或平移，文字缓慢浮现，体现高端家具的沉浸感。
- **高对比摄影**：收集专业摄影图（类似 Herman Miller 或 Steelcase 的光影感），展示家具的材质细节（木纹、金属质感、面料缝线）。

## 2. 内容逻辑：诗敏办公 (Seewin) 深度
- **场景化解决方案**：
  - **智慧教育**：未来教室、理化生实验室、VR/AR 创客空间。
  - **智慧办公**：协作区、专注区、行政办公、多功能会议室。
- **五星服务体系**：
  - **全链路流程**：需求调研 -> 方案策划 -> 3D 设计 -> 精益生产 -> 专业配送 -> 匠心安装 -> 终身售后。
  - **专业背书**：集中展示 ISO 认证（9001/14001/45001）、绿色环保认证、十环认证及各项外观/实用新型专利。

## 3. 配色整合：佐迪 (Zodi) 主色调
- **主色 (Primary)**：**深邃专业蓝 (Navy Blue)** —— 用于导航栏、核心标题、图标主体。象征科技、稳定与智能。
- **点缀色 (Accent)**：**活力科技橙 (Tech Orange)** —— 用于关键按钮 (CTA)、流程线段、重点参数。象征创新、服务热情与活力。
- **辅助色**：哑光银灰 (Silver Grey) + 温暖米白 (Cream White) —— 用于背景和次要色块，平衡冷色调。

## 4. 每日维护 (02:00 - 03:00) 核心任务清单
- **图片库扩充**：从 Behance、Pinterest 或专业建筑设计网站寻找相似风格的“场景图”作为灵感。
- **文案打磨**：将产品参数转化为“价值主张”（例：不是“桌高 75cm”，而是“符合亚洲人体工程学的黄金支撑高度”）。
- **SEO 深度部署**：针对“佛山智慧教育空间定制”、“智能人体工学桌椅厂家”进行关键词埋点。
