# 佐迪智能 (Zodi Smart) 网站优化方案

## 1. 核心定位梳理 (结合参考网站)
- **玛祖铭立 (Matsu.cn)**: 学习其**高端、极简、艺术化**的视觉表达。强调“空间美学”和“德系匠心”。
- **诗敏办公 (Seewin-edu.com)**: 学习其**教育场景化、全链路服务**的落地能力。强调“五星服务标准”和“专业教室解决方案”。

## 2. 优化建议清单

### A. 视觉与品牌 (向 Matsu 看齐)
- **大图叙事**: 首页采用高像素的实景或 3D 渲染图，展示“佐迪智能”如何改变传统办公/教学环境。
- **留白美学**: 减少冗余文字，让图片说话，体现专业感和高端感。
- **品牌故事**: 强调“佛山智造”与“智能交互”的结合，讲好佐迪自己的故事。

### B. 产品与方案 (向 Seewin 学习)
- **场景化分类**: 不要只罗列桌子椅子，要展示“未来教室”、“智慧办公”、“无人机实验室”等完整方案。
- **专业背书**: 明确展示 ISO 认证、专利证书、星级服务体系，建立客户信任。
- **全链路展示**: 从“咨询 -> 设计 -> 制造 -> 安装 -> 售后”全流程可视化。

### C. 交互与维护 (每日 02:00 - 03:00 任务)
- **动态更新**: 每日更新一个“案例实景”或“智能单品”。
- **性能优化**: 检查图片加载速度，优化 SEO 关键词（如：佛山智能家具、智慧校园解决方案）。

## 3. 下一步行动
